MESSAGE FROM CEO

古からある鉄 × 最先端のAI

鉄は人類が三千年以上にわたって扱ってきた素材です。私たちはその鉄を扱う商社として七十余年、お客様とサプライヤー様、そして社員とその家族の暮らしを支えてきました。これからも鉄は、私たちの仕事の中心であり続けます。

しかし、それを扱う方法は変わっていく必要があります。営業の進め方、見積の作り方、品質の見守り方、サプライヤーとのコミュニケーション。私たちの仕事のあらゆる場面に最先端のAIを取り込み、より良い形に変えていく。

その目的は効率化の先にあります。人が減っても、今いる仲間の負担を増やさないこと。誰かが育休に入っても、家族と過ごす時間が必要になっても、全員でカバーし応援できる組織であること。属人化を排除しAIに任せられることは任せる。それは共に働く仲間と、その家族の暮らしを守るための判断です。

古からある鉄を最先端のAIで扱う。それが東京管材の次の世代の姿です。

運用中

内示管理システム

// FOR MAJOR CLIENT A — TKK-NAIJI

大手取引先A社向けの内示データを一元管理する、自社開発の業務システム。発注から買取交渉、売上・利益分析までを一画面で扱える、専門商社のためのERPです。生成AIを駆使して自社で内製し、現場の声を反映しながら継続的に改善しています。

  • ダッシュボードで内示状況をリアルタイム可視化
  • 発注管理・買取交渉・売上利益分析を統合
  • マスタ管理機能で取引情報を一元化
  • 現場の業務フローに合わせたフルカスタム設計
運用中

週次集計・自動化ツール

// SALES DATA EXTRACTOR

特定取引先向けの週次売上明細を、ボタン一つで抽出・集計する自動化ツール。社内の売上データから対象顧客のレコードを抜き出し、整形済みのExcelとして出力。これまで手作業で行っていた定型集計を、数秒で完結させます。生成AIを駆使して内製。

  • 社内売上データをアップロードするだけで処理完了
  • 取引先別に整形・自動でシート分割
  • 週次の定型業務を大幅に短縮
  • 担当者の作業負担を軽減し、本来業務に集中
運用中

DX-OCR
注文書データ化

// PARTNERED — PAPER TO DATA

紙やPDFで届く注文書を、DX-OCRで瞬時にデータ化。手入力にかかっていた時間とミスをほぼゼロにし、データ化された情報をそのまま社内システムや基幹系へ連携できる体制を構築しています。外部パートナーと協業し導入。

  • 紙・PDF・FAXの注文書をAIで自動読取
  • 品番・数量・納期などの項目を構造化
  • 手入力ミスを大幅に削減
  • 社内システムへのデータ連携を効率化
運用中

配車最適化
システム

// LOGISTICS OPTIMIZER

受注・発注・在庫を一元管理し、月次配車計画と仕入先納期の調整を自動化する自社開発の業務システム。営業担当者の事務作業を軽減し、お客様・仕入先様との対話に集中できる環境を実現。生成AIを駆使して内製。

  • 受注残・発注残・在庫を統合し配車を自動立案
  • 共同配送と単独便のコスト最適な組み合わせ
  • 仕入先納期を学習し、欠品リスクを事前警告
  • 運送費の月次推移を可視化し、交渉資料として活用

VISION

変わらないもの
変えていくもの

創業から70余年、私たちは「お客様に適正な品質と価格の製品を提供する」という使命を掲げてきました。これからの時代にそれを果たすには、人の経験だけでもテクノロジーだけでも足りません。両方が掛け合わさったときにこそ、業界のスタンダードを超える価値が生まれると考えています。

そして、私たちは「働き方」そのものも、これからの時代に合わせて変えていきます。育休を取ること、家族と過ごす時間を大切にすること、誰かが休めば他のメンバーが支えること。AIに任せられる単純作業はAIに任せ、人にしかできない仕事に一人ひとりが集中する。それが、これからの東京管材の歩み方です。

QUALITY ASSURANCE / OUR 3 PILLARS

品質保証の 3つの柱

自社拠点での独自検査、不良が起きたときの徹底した要因解析、そしてAIの活用。3つの柱で、お客様への流出を未然に防ぎます。

01

検査体制の確立

海外サプライヤーへの定期指導と、自社物流拠点での独自検査。二段構えで流出を防ぎます。

02

品質不良対応

不良率の集計・現品確認・要因解析。鋳造工場経験のあるスタッフの助言も得ながら、根本的な不良要因の解決に取り組みます。

03

AIを活用した活動

品質保証の領域でもAI活用を進めています。検査記録のデータ化や、蓄積した技術ノウハウのAI化など、人の経験とAIを掛け合わせる取り組みを始めています。
